# A demo Node.js/Express app

The app consists of a single page with a two input fields and a button.

If valid input are provided, the correct sum will be computed and displayed.
Otherwise an error will be displayed.

# How to run

Clone the repo in a suitable directory and install dependencies:

```
cd node-demo
npm install
npm start
```

By default the app runs on port 3000. You may change this behavior by prefixing a different port value to the `npm start` command:

```
PORT=4000 npm start
```
